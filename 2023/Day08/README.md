# Day 8: Haunted Wasteland

You are navigating a network of nodes labeled AAA, ZZZ, etc.

## Part 1

**Task:** Follow a cyclic list of Left/Right instructions (e.g., `RL`). Start at `AAA`.

**Goal:** How many steps are required to reach `ZZZ`?
