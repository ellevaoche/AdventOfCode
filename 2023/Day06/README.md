# Day 6: Wait For It

You participate in toy boat races. You hold a button to charge the boat; speed increases with hold time, but move time decreases.

## Part 1

**Task:** Distance = (Time Held) * (Race Duration - Time Held).

**Goal:** For each race, determine the number of ways you can beat the record distance. Multiply these counts together.
