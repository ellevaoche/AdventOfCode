# Day 7: Camel Cards

You play a game similar to Poker with "Camel Cards".

## Part 1

**Task:** Order a list of hands based on strength (Five of a kind > ... > High card). If types are same, compare card labels lexicographically (A, K, Q, J, T, 9...2).

**Goal:** Find the rank of every hand. Calculate total winnings: Sum of (Rank * Bid).
