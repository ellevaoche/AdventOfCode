# Day 5: If You Give A Seed A Fertilizer

You need to map seeds to soil, soil to fertilizer, etc., until you reach a location.

## Part 1

**Task:** Convert each seed number through category maps (source-to-destination ranges) to find its corresponding location number.

**Goal:** What is the lowest location number that corresponds to any of the initial seed numbers?
