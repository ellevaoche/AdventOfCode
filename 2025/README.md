# Advent of Code 2025 - Solutions

This directory contains my solutions for the [Advent of Code 2025](https://adventofcode.com/2025) challenges.

These solutions were originally part of a shared exchange repository and have been consolidated here.

## Structure

Each day is organized into its own folder:


- `DayXX/`
  - `solution.lua` (or similar): The solution code.
  - Test inputs or other assets.
