# Advent of Code 2024 - Solutions

This directory contains my solutions for the [Advent of Code 2024](https://adventofcode.com/2024) challenges.

## Structure

Each day is organized into its own folder:

- `DayXX/`
  - `solution.go`: The solution code in Go.
  - `test.txt`: Example input for testing.
  - `README.md`: Problem description and approach.

## Language

All solutions for 2024 are implemented in **Go**.
